# MULER ART STUDIO | Barbería Murcia

Web oficial de MULER ART STUDIO. Barbería premium y estudio de arte masculino en el centro de Murcia.

## Ejecución Local

**Prerrequisitos:** Node.js (v18 o superior)

1. **Instalar dependencias:**
   Ejecuta el siguiente comando en tu terminal:
   ```bash
   npm install
   ```
   *(Asegúrate de escribir **npm**, no "nom")*

2. **Configurar API Key:**
   Establece tu `GEMINI_API_KEY` en el archivo `.env.local`.

3. **Arrancar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

## Despliegue

Si utilizas Vercel o plataformas similares, los comandos detectados automáticamente deberían ser:
- **Build Command:** `npm run build`
- **Install Command:** `npm install`
- **Output Directory:** `dist`
